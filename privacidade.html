Politica de Privacidade - CineLancamentos

Ultima atualizacao: 21 de maio de 2025

Bem-vindo ao CineLancamentos! Desenvolvido por Meigston Servicos Tech.

1. Informacoes Coletadas

O aplicativo acessa dados publicos da API TMDb para exibir informacoes sobre filmes. Ao utilizar Google AdMob, dados como identificadores de dispositivo podem ser coletados para exibicao de anuncios.

2. Publicidade (Google AdMob)

Este aplicativo usa Google AdMob. Para usuarios no EEE, Reino Unido e Suica, sera solicitado consentimento antes de anuncios personalizados (GDPR). Gerencie em: google.com/settings/ads. Politica Google: policies.google.com/privacy

3. Permissoes

  • Internet: para dados dos filmes e anuncios.
  • ID publicidade: usado pelo AdMob com consentimento.

4. Servicos Terceiros

  • TMDb: Politica de Privacidade
  • Google AdMob: Politica Google

5. Direitos GDPR

Usuarios do EEE/Reino Unido/Suica tem direito de acesso, retificacao, exclusao, restricao, portabilidade e oposicao. Contate-nos para exercer esses direitos.
